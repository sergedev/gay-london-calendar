# Gay London Calendar

A curated calendar of LGBTQ+ events across London — community gatherings,
fitness meetups, board game nights, food clubs, Pride events, and the
city's queer festivals. Built to surface the events worth knowing about,
without the daily-club-listings firehose.

**Live:** https://sergedevelops.github.io/gay-london-calendar/

## Stack

Static site — vanilla JS + Tailwind CDN, no build step. Event data lives
in `data/*.json` and is regenerated by Python scrapers in `scripts/`.

```
.
├── index.html       # entry point
├── app.js           # renderer (list, month, year, drawer)
├── assets/          # source logos + event hero images
├── data/            # sources.json + one JSON per source
└── scripts/         # per-source refresh + enrich scripts
    └── _merge.py    # idempotent merge helper (preserves custom fields)
```

## Run locally

```bash
python3 -m http.server 8000
open http://localhost:8000
```

## Refresh data

To refresh every source in one go:

```bash
python3 scripts/refresh-all.py
```

This discovers and runs every `scripts/refresh-*.py` and then every
`scripts/enrich-*.py` in sequence. A failing script doesn't abort the
run — you get a pass/fail summary at the end.

To refresh a single source on its own:

```bash
python3 scripts/refresh-village-drinks.py
python3 scripts/refresh-bgo-mixer.py
# ...etc
```

All scripts are **idempotent** — re-running them won't wipe
non-script-managed fields on events (category overrides, manual edits).
See `scripts/_merge.py` for the contract.
